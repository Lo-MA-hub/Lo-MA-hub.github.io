PIPELINE

From pixels to a navigable scene

  1. ParseMulti-task CNN recognizes rooms, walls, and icons.
  2. VectorizeWall centerlines, openings, and furniture are converted to geometry.
  3. ReconstructExtruded geometry creates an interactive 3D interior model.

About live uploads

This public site includes the finished interactive viewer. Live processing of a new plan requires the CubiCasa5K model to run on a separate inference service; GitHub Pages cannot execute PyTorch models.
